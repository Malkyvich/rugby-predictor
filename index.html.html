🔧 Firebase Setup Required

To enable automatic syncing between all players, please set up Firebase (takes 5 minutes, free forever):

  1. Go to Firebase Console
  2. Click "Add project" → Name it "rugby-predictor" → Continue
  3. Disable Google Analytics (not needed) → Create project
  4. Click "Realtime Database" in left menu → Create Database
  5. Choose location (europe-west1) → Start in test mode → Enable
  6. Click ⚙️ (Settings) → Project settings → Scroll to "Your apps"
  7. Click "</>" (Web app) → Register app → Copy the config
